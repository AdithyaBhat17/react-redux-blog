This repository contains the source code for the blog built during React-Redux workshop held at Betsol from March 9th to 12th, 2020.

## Installation guide

* Clone this repository - `git clone https://github.com/AdithyaBhat17/react-redux-blog.git`
* go to the project folder - `cd react-redux-blog`
* Install the required dependencies - `npm install`
* Create a .env file and add the required credentials
* Start the development server - `npm start`
